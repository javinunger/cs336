Lab 3 Answers:

3.1

	- Express.js is a web framework for Node.js, and Node.js is a JavaScript runtime
	  built on V8.

	- The packages.json file holds metadata for the relevant project. NPM uses this 
	  file to get information so that it knows what the project is and what its 
	  dependencies are.

3.2

	- "Static" files are files that are just kinda there.

	- The server does not continue to serve up the hello world message, likely because
	  it hasn't been told to do differently. The console prints it once then it's done.
