Answer to Lab 03 Exercises:

  • 3.1

    • Express.js is a web framework for Node.js, and Node.js is a JavaScript runtime built on V8.
    • The packages.json file holds metadata for the relevant project. NPM uses this file to get information so that it knows what the project is and what its dependencies are.

  • 3.2

    • "Static" files are files that are just kinda there - they're loaded and won't change until the page reloads.
    • The server does not continue to serve up the hello world message, since index.html is being served up instead.
